# 💰 FinTrack Pro

A modern Finance Tracker web application built using **HTML, CSS, and JavaScript**. It helps users manage income and expenses, monitor cash flow, and track financial transactions with a clean and responsive dashboard.

## ✨ Features

- 🔐 User Registration & Login
- 💸 Add Income & Expense Transactions
- ✏️ Edit Existing Transactions
- 🗑️ Delete Transactions
- 📊 Dynamic Cash Flow Analysis Chart
- 📈 Financial Summary Cards
- 🔍 Search Transactions
- 🎯 Filter by Income / Expense
- 🌙 Dark Mode
- 💱 Multiple Currency Support (USD, INR, EUR, GBP, JPY)
- 👤 User Profile Settings
- 💾 Local Storage Support
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Font Awesome
- Local Storage API

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/FinTrackPro.git
```

### Open the Project

```bash
cd FinTrackPro
```

Open **index.html** in your browser.

---

## 🎯 How to Use

1. Register a new account.
2. Login using your credentials.
3. Add income or expense transactions.
4. View updated cards and cash flow chart.
5. Search or filter transactions.
6. Edit or delete transactions.
7. Change currency and dark mode from Settings.
8. Logout securely.

---

## 📊 Dashboard Includes

- Current Balance
- Total Income
- Total Expense
- Total Transactions
- Cash Flow Analysis Chart
- Transaction History
- Search & Filter
- Settings Panel

---

## 💾 Data Storage

This project uses **Local Storage** to store:

- User Credentials
- Transactions
- Selected Currency
- Dark Mode Preference

No backend or database is required.
